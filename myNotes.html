HTML DEFINIÇÃO - Hypertext Markup Language



É o arquivo utilizado pelos browsers para estabelecer contato com os servidores.

Processo: >> NAVEGADOR >> INTERNET >> PROTOCOLO HTML >> SERVIDOR


HTML ELEMENTOS

O arquivo HTML é definido basicamente por elementos , tags, "Inline" e "block", quase todas possuem o mesmo critério de "abertura" e "fechamento", com algumas exceções, como a tag "br" e "img", que não precisam de fechamento ao final da chamada de comando.


HTML ATRIBUTOS


Todos os elementos HTML podem possuir atributos. Geralmente são atribuidos logo ao início da tag, como por ex: "img src = ..." onde src é o atributo da tag img e o seu "valor" vem em seguida.

Para saber mais sobre os atributos clique aqui.


HTML ESTRUTURAÇÃO


Importante lembrar que escrevemos os códigos para os computadores, mas também para outras pessoas. Sendo assim, vale ressaltar que o "recuo" representa a ordem das idéias: quanto maior o recuo, mais recente a idéia!!!
